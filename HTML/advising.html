There are 3 ways to add or drop a class here at MNU.

  1. Visit the Lunn Building on campus and fill out the form given to you in the Registrar's office.
  2. Visit this link to fill out the class add and drop form and follow the instructions given.
  3. Do it completly online
    1. Go to my.mnu.edu
    2. Select "Self-Service Banner".
    3. Select "Enter Secure Area - Login" at the top.
    4. Login with your MNU email and password.
    5. Select "Student Services".
    6. Select "Registration".
    7. Select "Add or Drop Classes".
    8. Select the semester in which you would like to drop or add a class.
    9. Read through and accept the terms and conditions.
    10. If you are dropping a class, find the class in the list under "Current Schedule". Under the "Action" Column, select the drop down arrow and select "Web Drop". Then at the bottom of the page, select "Submit Changes".
    11. If you are adding a class, enter the CRNs number for the class you would like to add to your schedule under "Add Classes Worksheet". Once you have done that, select "Submit Changes" at the bottom of the page.
